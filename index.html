Basic Text Formatting & Semantic Tags

About Semantic Tags

Semantic tags help describe the meaning of content on a webpage.

Examples include header, main, and footer.

Using semantic structure is important for accessibility.

Inline Formatting Examples

A word can be strongly emphasized using the strong tag.

Text can be slightly emphasized using the em tag.

You can also highlight text using the mark tag.

Italic text and bold text are also used.

Programming Examples

HTML code is wrapped in tags like <p> to create paragraphs.
CSS can style elements using selectors like h1 or .classname.
JavaScript can manipulate the DOM with functions like document.querySelector().
Developers learn these piece by piece.
Practice helps build knowledge.
